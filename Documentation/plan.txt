Users config all options in 'config.yml'

Config:
    proxmox (config for the proxmox host):
        ip for host
        username
        password
        node_name

    docker (config for using docker containers):
        ip for host
        username
        password

    widgets (config for the dashboard widgets):
        monitor (view status of webui's, include restarting button to proxmox host/docker container):
            name
            ip url
            type (proxmox, docker, custom)
            proxmox:
                vm or lxc
                wm/lxc id
            docker:
                docker container name
            custom (executes custom command on any host):
                command
                ip of host
                username (for ssh)
                password (for ssh)

Structure:
    \-Functions (code for each widget)
      \-widget1
      \-widget2
      \-widget3
    \-Graphics (elements for the gui)
      \-picture1
      \-picture3
      \-picture2
